## 知見

- React の`useEffect`を使いたいときは` watchEffect`を使うと良さげ?
- `<script setup>`を使うことで`setup()`に記述していた内容をより js らしく書ける
  - return もいらないので楽
  - vetur は vue3 の一部記法に対応していないので volar を使用した方が良い
  - script タグと併用できるので一度のみ呼び出したい処理がある場合は、script に記述する
